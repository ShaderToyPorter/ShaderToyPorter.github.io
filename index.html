Info
This tool is made to export mods from https://shadertoy.com to a Geometry Dash mod made by Mat (@mateus44_ on twitter) that allows you to replace the menu background with glsl shaders.

If you run into any issues with this tool contact me on discord tnings#6969

How to use
1. Find a shader you want to port from shadertoy.com
2. Copy/paste the code on the right into the left textbox or into a txt file on your computer, and drag it into the dashed box
3. Click download!

By default the filename will be menu-shader.fsh, but you can set a custom name by entering it into the textbox to the right of the download button

FAQ
GD crashes when I launch it after using the exported file
This is most likely this is caused because the shader you are trying to port uses textures, Mat's mod doesn't support textures. (the shader is incompatible, you can't do much about that)
